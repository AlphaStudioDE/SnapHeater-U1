# SnapHeater U1 iOS App

iOS is a planned target for the SnapHeater U1 companion app.

The first public mobile work is the Android app shell under `apps/android/SnapHeaterU1`. The iOS implementation should follow the same product structure:

- Status-first dashboard.
- Mode selection with Safe stop always visible.
- Safety checklist mirroring firmware validation state.
- Diagnostics for BLE and firmware bring-up.
- EN primary UI with PL and DE localization.

No iOS build project is committed yet.
