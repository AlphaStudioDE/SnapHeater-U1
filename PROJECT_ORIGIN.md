# Project Origin and Attribution

SnapHeater U1 is an original firmware framework and development skeleton created by **Damian Borkowski** for Panda Breath-style ESP32-C3 chamber-heater hardware, with a dedicated focus on **Snapmaker U1** integration through Moonraker.

This project is intended as the original upstream repository for the SnapHeater U1 firmware concept and its related development work.

## Original project identity

- **Project name:** SnapHeater U1
- **Original author:** Damian Borkowski
- **GitHub handle:** `@damianborkowski88`
- **Primary target:** Snapmaker U1 + Panda Breath-style ESP32-C3 chamber-heater hardware
- **Project type:** Open firmware framework / development skeleton
- **Contest context:** Created as a Snapmaker U1 ecosystem project suitable for submission to the Snapmaker Innovation Fund / Open Competition.
