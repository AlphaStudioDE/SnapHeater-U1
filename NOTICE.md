# Notice

SnapHeater U1

Original project by Damian Borkowski (`@damianborkowski88`).

SnapHeater U1 is an original open firmware framework for Panda Breath-style ESP32-C3 chamber-heater hardware, designed around Snapmaker U1 cooperation through Moonraker.

This project was created as a Snapmaker U1 ecosystem project suitable for public open-source development and contest/community submission.
